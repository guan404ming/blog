## 📌 [GoodbyeDPI — Deep Packet Inspection circumvention utility](https://google.com)

#### 簡介

GoodbyeDPI 是一款設計用於繞過許多網際網路服務供應商 (ISP) 中使用的 Deep Packet Inspection (DPI) 系統的軟體，這些系統會封鎖對特定網站的訪問。它處理使用光纖分路器或端口鏡像 (**被動 DPI**) 連接的 DPI，這些 DPI 不會封鎖任何資料，但僅比請求的目的地更快地回复，以及按順序連接的 **主動 DPI**。

#### 主要功能

* 繞過被動 DPI 和主動 DPI，使您能夠訪問被封鎖的網站。
* 支援多種技術，例如 TCP 分片、HTTP 標頭修改和偽造 HTTP/HTTPS 封包，以欺騙 DPI。
* 可自訂設定，允許您根據自己的網路條件調整繞過 DPI 的方法。
* 支援將 GoodbyeDPI 作為 Windows 服務安裝，以確保它在背景中持續運行。

#### 如何使用

1. 從 [Releases 頁面](https://github.com/ValdikSS/GoodbyeDPI/releases) 下載最新版本。
2. 解壓縮檔案並運行適當的腳本：
    * **俄羅斯：** `1_russia_blacklist_dnsredir.cmd`
    * **其他國家：** `2_any_country_dnsredir.cmd`
3. 這些腳本會啟動 GoodbyeDPI，並將 DNS 解析器重新導向到 Yandex DNS 的非標準端口 (以防止 DNS 污染)。
4. 如果有效，您可以直接使用 GoodbyeDPI 或者進一步配置它，例如使用 `--blacklist` 選項，如果您的國家/地區有已知的被封鎖網站列表。

    您可以通過運行 `goodbyedpi.exe` 可執行檔並查看是否能夠訪問被封鎖的網站來檢查您的 ISP 的 DPI 是否可以被繞過。
## 📌 Teable: Postgres-Airtable Fusion

#### 簡介

Teable 是一個基於 PostgreSQL 的超高速、實時、專業、開發者友善、無需代碼的數據庫。它使用簡單的電子表格界面來創建複雜的企業級數據庫應用程序。透過無需代碼的設計，輕鬆解鎖高效的應用程序開發，擺脫數據安全和可擴展性的障礙。

#### 主要功能

* **Spreadsheet-like Interface:** 提供類似電子表格的界面，包括單元格編輯、公式支持、數據排序和過濾、聚合函數、數據格式化、分組、凍結列、導入/導出功能、行樣式和條件格式 (即將推出)、圖表和可視化工具 (即將推出)、數據驗證 (即將推出)、撤銷/重做 (即將推出)、註釋和批註 (即將推出) 等功能。
* **Multiple Views:**  提供多種視圖，讓使用者可以根據需求以不同方式視覺化和互動數據，包括網格視圖、表單視圖、看板視圖、日曆視圖 (即將推出)、畫廊視圖 (即將推出)、甘特圖視圖 (即將推出)、時間軸視圖 (即將推出)。
* **Super Fast:** 提供驚人的響應速度和數據容量，輕鬆處理數百萬數據，並提供自動數據索引以達到最大速度，支持一次性批量數據操作。
* **Full-featured SQL Support:**  無縫整合您熟悉的軟體，包括 BI 工具 (例如 Metabase PowerBi...)、無需代碼工具 (例如 Appsmith...)，以及使用原生 SQL 直接取回數據。

#### 如何使用

1. **快速體驗:** 選擇 [模板中心](https://template.teable.io) 中的場景，並點擊「使用此模板」。
2. **追求高性能:** 嘗試 [100 萬行演示](https://app.teable.io/share/shrVgdLiOvNQABtW0yX/view) ，感受 Teable 的速度。
3. **快速學習:** 點擊 [教學](https://help.teable.io/quick-start/build-a-simple-base) 快速學習如何使用 Teable。
4. **自行部署:** 點擊 [在 Railway 部署](https://railway.app/template/wada5e?referralCode=rE4BjB) 自行部署 Teable。
## 📌 ComfyUI

#### 簡介

ComfyUI 是一個強大的且可模組化的 Stable Diffusion 圖形使用者介面 (GUI) 和後端，提供圖表/節點/流程圖介面來設計和執行進階的 Stable Diffusion 管線。 

#### 主要功能

* **節點/圖表/流程圖介面:**  讓使用者無需編碼即可建立複雜的 Stable Diffusion 工作流程。
* **支援多種模型:** 包含 SD1.x、SD2.x、SDXL、Stable Video Diffusion、Stable Cascade、SD3 和 Stable Audio。
* **其他功能:**
    * Flux
    * 非同步佇列系統
    * 多項最佳化
    * 智慧型記憶體管理
    * 支援 CPU 運算
    * 載入 ckpt、safetensors 和 diffusers 模型
    * 嵌入式/文字反轉
    * Loras、Hypernetworks
    * 載入/儲存工作流程
    * 繪製區域、修復、Inpainting
    * ControlNet 和 T2I-Adapter
    * Upscale Models (ESRGAN、SwinIR 等)
    * unCLIP Models
    * GLIGEN
    * 模型合併
    * LCM 模型和 Loras
    * SDXL Turbo
    * AuraFlow
    * HunyuanDiT
    * 潛在預覽
    * 快速啟動
    * 離線運作

#### 如何使用

* **安裝:**
    * **Windows:** 從 [Releases 頁面](https://github.com/comfyanonymous/ComfyUI/releases) 下載並解壓縮可攜式獨立版本。
    * **Jupyter Notebook:** 使用 [Jupyter Notebook](notebooks/comfyui_colab.ipynb) 在 Paperspace、Kaggle 或 Colab 等服務上執行。
    * **手動安裝:** 
        1. 使用 Git 克隆此 repo。
        2. 將 SD 模型放在 `models/checkpoints` 資料夾中。
        3. 將 VAE 放在 `models/vae` 資料夾中。
        4. 安裝相應的 Python 套件。
* **執行:** 在終端機中執行 `python main.py`。
* **使用快捷鍵:**
    * Ctrl + Enter: 佇列當前圖表以進行生成。
    * Ctrl + Shift + Enter: 佇列當前圖表為第一個以進行生成。
    * Ctrl + Z/Ctrl + Y: 撤銷/重做。
    * Ctrl + S: 儲存工作流程。
    * Ctrl + O: 載入工作流程。
    * Ctrl + A: 選取所有節點。
    * Alt + C: 折疊/展開選定節點。
    * Ctrl + M: 靜音/取消靜音選定節點。
    * Ctrl + B: 繞過選定節點。
    * Delete/Backspace: 刪除選定節點。
    * Ctrl + Backspace: 刪除當前圖表。
    * Space: 移動畫布。
    * Ctrl/Shift + Click: 將點選的節點新增到選取項目中。
    * Ctrl + C/Ctrl + V: 複製和貼上選定節點。
    * Shift + Drag: 同時移動多個選定節點。
    * Ctrl + D: 載入預設圖表。
    * Alt + `+`: 畫布縮放。
    * Alt + `-`: 畫布縮小。
    * Ctrl + Shift + LMB + Vertical drag: 畫布縮放。
    * Q: 切換佇列的顯示狀態。
    * H: 切換歷史記錄的顯示狀態。
    * R: 重新整理圖表。
    * 雙擊 LMB: 開啟節點快速搜尋面板。
## 📌 JMusicBot

#### 簡介

JMusicBot 是一個跨平台的 Discord 音樂機器人，擁有乾淨的介面，易於設定和運行。

#### 主要功能

* 易於運行（只要確保已安裝 Java，並運行！）
* 快速載入歌曲
* 不需要外部金鑰（除了 Discord Bot token）
* 流暢的播放
* 伺服器特定的設定，為「DJ」角色提供音樂管理權限
* 清潔且美觀的選單
* 支援多個網站，包括 Youtube、Soundcloud 等
* 支援多個線上廣播/串流
* 支援本地檔案
* 支援播放清單（包括網路/youtube 和本地）

#### 如何使用

請參考 [Setup Page](https://jmusicbot.com/setup) 頁面，了解如何自行運行這個機器人。

#### 主要功能

JMusicBot 支援 [lavaplayer](https://github.com/sedmelluq/lavaplayer#supported-formats) 所支援的所有來源和格式：

* **來源:**
    * YouTube
    * SoundCloud
    * Bandcamp
    * Vimeo
    * Twitch Streams
    * 本地檔案
    * HTTP URLs
* **格式:**
    * MP3
    * FLAC
    * WAV
    * Matroska/WebM (AAC, Opus 或 Vorbis 編碼器)
    * MP4/M4A (AAC 編碼器)
    * OGG Streams (Opus, Vorbis 和 FLAC 編碼器)
    * AAC Streams
    * 串流播放清單 (M3U 和 PLS)
