## 📌 [toss/es-toolkit](https://github.com/toss/es-toolkit)
<Callout>
    Description: A modern JavaScript utility library that's 2-3 times faster and up to 97% smaller—a major upgrade to lodash.\
    🌐 TypeScript｜⭐️ 5,727
</Callout>
    #### 簡介

* es-toolkit 是一個高效能的 JavaScript 工具庫。
* 包含小型捆綁包和強大的型別註解。
* 提供現代實作的各種日常實用功能，例如 debounce、delay、chunk、sum 和 pick。
* 旨在提升效能，在現代 JavaScript 環境中可提升 2-3 倍的效能。
* 支援 tree shaking，與其他庫相比，可將 JavaScript 程式碼減少高達 97%。
* 包含內建的 TypeScript 支援，提供直觀且強大的型別。
* 提供有用的型別守衛，例如 isNotNil。
* 經過嚴格測試，覆蓋率達 100%，確保可靠性和穩定性。

#### 主要功能

* Debounce
* Delay
* Chunk
* Sum
* Pick
* Tree shaking
* TypeScript support
* Type guards

#### 如何使用

* 使用 `import` 語法從 `es-toolkit` 匯入所需的功能。
* 例如，使用 `import { debounce, chunk } from 'es-toolkit';` 匯入 debounce 和 chunk 功能。
* 根據需要使用匯入的功能。
* 例如，使用 `const debouncedLog = debounce(message => { console.log(message); }, 300);` 宣告一個 debouncedLog 變數，並使用 debounce 功能將 message 訊息的日誌輸出延遲 300 毫秒。
* 然後可以使用 `debouncedLog('Hello, world!');` 呼叫 debouncedLog 變數，以輸出 debounced 的訊息。

<Callout>
    Description: Various AI scripts. Mostly Stable Diffusion stuff.\
    🌐 Python｜⭐️ 1,697

* 該工具箱基於研究，並且會定期更新。
* 如果遇到問題，請檢視之前的提交。
* 需要 Python 3.10 以上版本、Nvidia GPU 和 Python venv。
* 該工具箱需要安裝 Git 和必要的套件。

#### 主要功能

* 訓練 FLUX.1 模型，需要至少 24GB 的 VRAM。
* 使用 FLUX.1-dev 和 FLUX.1-schnell 模型。
* 訓練 LoRA 和 LoCON 模型。
* 生成批次圖片。
* 提取 LoRA 和 LoCON 模型。
* 調整 LoRA 模型的權重。
* 訓練 LoRA 滑桿。
* 擴充套件功能，允許開發者自訂功能。
* 模型合併功能。

* 克隆 GitHub 儲存庫。
* 建立虛擬環境並安裝必要的套件。
* 複製範例設定檔並編輯。
* 執行 `python run.py config/whatever_you_want.yml` 進行訓練。

* 準備資料集，包括圖片和文字檔。
* 使用 `huggingface-cli login` 登入 Hugging Face。
* 使用 `modal run run_modal.py --config-file-list-str=/root/ai-toolkit/config/whatever_you_want.yml` 在 Modal 上訓練模型。
* 使用 `modal volume get flux-lora-models your-model-name` 下載訓練好的模型。
* 參考範例設定檔和檔案。
* 加入 Discord 社群尋求幫助。
## 📌 [lllyasviel/stable-diffusion-webui-forge](https://github.com/lllyasviel/stable-diffusion-webui-forge)
<Callout>
    
    🌐 Python｜⭐️ 6,622
</Callout>
    #### 簡介

- Stable Diffusion WebUI Forge 是一個基於 Stable Diffusion WebUI (Gradio) 的平臺，旨在簡化開發、最佳化資源管理、加速推理和探索實驗性功能。
- “Forge” 的命名靈感來自於 “Minecraft Forge”。這個專案旨在成為 SD WebUI 的 Forge。
- Forge 目前基於 SD-WebUI 1.10.1。由於原版 SD-WebUI 幾乎靜止不動，Forge 將每 90 天或是在出現重要修復時與原版 WebUI 同步。

#### 主要功能

- Gradio 4 UI
- Flux Tutorial (BitsandBytes Models, NF4, "GPU Weight", "Offload Location", "Offload Method", etc)
- LoRA 管理
- LayerDiffuse 透明影象編輯
- ControlNet 整合
- Flux BNB NF4 / GGUF Q8_0/Q5_0/Q4_0/Q4_1 原生支援
- UnetPatcher
- 各種擴充套件支援 (Adetailer, etc)
- Wacom 128 級觸控壓力支援
- API 終端

#### 如何使用

- 使用一鍵安裝包安裝 Forge (包含 Git 和 Python)。
- 解壓縮後，使用 update.bat 更新，使用 run.bat 執行。
- 執行 update.bat 很重要，否則可能會使用舊版本，存在潛在的錯誤。
- 也可以使用 Git 和 Python 手動安裝 Forge。
- 可以下載舊版本。
- 如果遇到問題，可以提交問題，開發者會定期檢視。
- 如果“Forge Status”沒有更新，則表示無法復現任何問題，重新安裝應該可以解決大多數問題。
- UnetPatcher 提供所有程式碼來實現 FreeU V2。
- 檢視 extension-builtin/sd_forge_freeu/scripts/forge_freeu.py 檔案以獲取更多資訊。
